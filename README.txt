CLO3D Planner — نسخه کامل

نحوه اجرا روی لپ‌تاپ (بدون هاست):
1) همه فایل‌ها را در یک پوشه (clo3d-planner) ذخیره کن.
2) برای بهترین نتیجه یک «سرور محلی» ساده راه بینداز (لازم است تا سرویس‌ورکر فعال شود):

Windows/macOS/Linux:
- اگر Python داری:
  - داخل پوشه پروژه ترمینال/Command Prompt باز کن و اجرا کن:
    برای پایتون۳:
      python -m http.server 8080
    سپس در مرورگر برو به:
      http://localhost:8080
- یا با Node.js:
  npm install -g serve
  serve .

3) اگر فقط روی فایل دوبارکلیک کنی هم کار می‌کند، اما ممکن است سرویس‌ورکر فعال نشود.
4) «پشتیبان JSON» را هفتگی بگیر.

روی آیفون (نسخه وب آفلاین):
- پوشه را روی یک هاست رایگان HTTPS (GitHub Pages/Netlify) آپلود کن.
- لینک را در Safari باز کن → Share → Add to Home Screen.
- از این پس آفلاین هم کار می‌کند (به لطف SW).
